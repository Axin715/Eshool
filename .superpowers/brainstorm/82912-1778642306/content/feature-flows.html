核心功能流程设计

5 个核心功能的操作流程与关键状态

📝 请假申请

操作人：家长（为自己孩子）/ 老师（为任教班级学生）
流程：选择学生 → 选择请假类型（事假/病假/其他）→ 选择日期时段 → 填写原因 → 提交申请 → 等待审批
状态：待审批 → 已批准 / 已拒绝
关键点：老师替学生请假时需要先选择班级再选择学生

🎫 访校申请

操作人：家长 / 老师
类型：短期访校（单次）+ 长期访校（多次有效）
流程：选择类型 → 填写访校事由 → 选择日期/时段 → 提交申请 → 审批通过 → 生成访校二维码
关键点：进校时展示二维码由门卫扫码验证；长期访校码有有效期

💚 心理校园

功能：预约校医 / 预约心理咨询
流程：选择预约类型（校医/心理咨询）→ 查看可预约时段 → 填写简要描述 → 提交预约 → 确认
关键点：展示已预约记录；支持取消预约；每个时段有人数限制

💳 饭卡充值

操作人：家长（为自己孩子充值）/ 老师（为任教班级学生充值）
流程：选择学生 → 查看当前余额 → 选择充值金额（固定档位/自定义）→ 模拟支付 → 充值到账
关键点：Mock 支付流程，模拟完整的充值体验；显示充值记录

📅 班级值班排班

操作人：仅班主任
流程：选择日期（周视图）→ 为每日添加值班学生 → 可拖拽调整 → 发布排班表
关键点：班主任身份校验；班级学生列表选择；已发布的排班表全班可见（消息推送）；支持按周查看和修改
