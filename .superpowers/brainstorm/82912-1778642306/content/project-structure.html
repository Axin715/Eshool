项目目录结构

uni-app (Vue 3 + TypeScript) 标准项目结构

src/
├── pages/
│ ├── login/ 登录页（微信授权 + 验证码 + 身份选择）
│ ├── home/ 主页 Tab（新闻轮播、活动公告、快捷入口）
│ ├── messages/ 消息 Tab（通知/动态/系统消息）
│ ├── features/ 功能 Tab（功能入口列表）
│ │ ├── leave/ 请假申请 + 审批
│ │ ├── visit/ 访校申请 + 二维码
│ │ ├── psychology/ 心理校园预约
│ │ ├── meal/ 饭卡充值
│ │ └── duty/ 班级值班排班（班主任）
│ └── profile/ 我的 Tab（信息 + 身份切换 + 记录）
├── components/ 通用组件
│ ├── news-card
│ ├── message-item
│ ├── student-picker
│ └── qr-code-display
├── stores/ Pinia 状态管理
│ ├── user.ts 用户信息 + 身份
│ ├── message.ts 消息列表
│ └── app.ts 全局状态
├── services/ API 服务层
│ ├── mock/ Mock 数据
│ └── api/ API 接口定义
├── utils/ 工具函数
│ ├── request.ts 请求封装
│ ├── auth.ts 权限判断
│ └── constants.ts 常量定义
├── static/ 静态资源
├── App.vue
├── main.ts
├── pages.json
├── uni.scss
└── manifest.json
